## Projeto Final de Machine Learn ESPM
### Objetivo:
Atingir um nivel de acurácia de ao menos 80% com qualquer um dos pipelines desenvolvidos.

### Resultados:
- Gradient Boosting Classifier -> 70%
- Random Forest Classifier -> 68%
- KNN -> 62%

O melhor resultado alcançado foi 70% de acurácia utilizando o algoritmo Gradient Boosting Classifier
